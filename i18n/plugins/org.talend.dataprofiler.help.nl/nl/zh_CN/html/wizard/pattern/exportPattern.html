导出模式：

此向导将多个正则表达式导出到 csv 文件中。

文件格式如下：
  • C1：标签：模式的标签（不得为空）
  • C2：目的：模式的目的（可以为空）
  • C3：描述：模式的描述（可以为空）
  • C4：作者：正则表达式的作者（可以为空）
  • C5：相对路径：根文件夹的相对路径（可以为空）
  • C6：All_DB_Regexp：适用于所有数据库的正则表达式（可以为空）
  • C7：DB2_Regexp：适用于 DB2 数据库的正则表达式（可以为空）
  • C8：MySQL_Regexp：适用于 MySQL 数据库的正则表达式（可以为空）
  • C9：Oracle_Regexp：适用于 Oracle 数据库的正则表达式（可以为空）
  • C10：PostgreSQL_Regexp：适用于 PostgreSQL 数据库的正则表达式（可以为空）
  • C11：SQL_Server_Regexp：适用于 SQL Server 数据库的正则表达式（可以为空）
  • C12：Sybase_Regexp：适用于 Sybase 数据库的正则表达式（可以为空）
  • C13：Ingres_Regexp：适用于 Ingres 数据库的正则表达式（可以为空）
  • C14：Informix_Regexp：适用于 Informix 数据库的正则表达式（可以为空）
  • C15：SQLite3_Regexp：适用于 SQLite3 数据库的正则表达式（可以为空）
  • C16：Teradata_Regexp：适用于 Teradata 数据库的正则表达式（可以为空）

注意：
导出的 csv 文件可用于将模式导入另一个 Talend Open Studio for Data Quality。
