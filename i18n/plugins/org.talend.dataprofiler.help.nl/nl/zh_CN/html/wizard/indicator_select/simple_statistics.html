基础统计

它们提供关于记录数量的基础统计。

它会提供列内容概览、存在的 null 数据数量、不同数据的数量...

  • 空白计数：对空白行数量进行计数。“空白”是仅包含空格的非 null 文本数据。请注意，Oracle 不区分空字符串和 null 值。

  • 默认值计数：对默认值的数量进行计数。

  • 不同计数：对列中的不同值数量进行计数。

  • 重复计数：对出现次数超过一次的值数量进行计数。存在以下关系：重复计数 + 唯一计数 = 不同计数。

  • Null 计数：对 null 行的数量进行计数。

  • 行计数：对行数进行计数。

  • 唯一计数：对只出现一次的不同值数量进行计数。它必然小于或等于不同计数。

例如，a,a,a,a,b,b,c,d,e => 9 个值，5 个不同值，3 个唯一值，2 个重复值。
