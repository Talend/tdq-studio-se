シンプル統計

レコードの数にシンプル統計が提供されます。

カラムのコンテンツ、nullデータの数、重複を除いたデータの数、その他の概要が示されます。

  • 空白カウント: 空白の行数をカウントします。「空白」とはnullではないテキストデータで、空白のみが含まれるものです。Oracleは空の文字列とnull値を区別しません。

  • デフォルト値カウント: デフォルト値の数を数えます。

  • 重複を除いたカウント: カラム内の重複を除いた値の数をカウントします。

  • 重複カウント: 2回以上出現する値の数をカウントします。次の関係があります: 重複カウント + ユニークなカウント = 重複を除いたカウント。

  • NULL値カウント:nullの行数をカウントします。

  • 行カウント:行数をカウントします。

  • ユニークカウント:1回しか出現しない重複を除いた値の数をカウントします。これは重複を除いたカウント以下になります。

たとえば、a、a、a、a、b、b、c、d、e =>値9、重複を除いた値5、ユニーク値3、重複値2となります。
