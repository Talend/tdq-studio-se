パターンのエクスポート:

このウィザードでは、複数の正規表現がcsvファイルにエクスポートされます。

ファイル形式は次のとおりです。
  • C1: Label: パターンのラベル(必須)
  • C2: Purpose: パターンの目的(任意)
  • C3: Description: パターンの説明(任意)
  • C4: Author: 正規表現の作成者(任意)
  • C5: Relative_Path: ルートフォルダーへの相対パス(任意)
  • C6: All_DB_Regexp: 全てのデータベースに適用可能な正規表現(任意)
  • C7: DB2_Regexp: DB2データベースに適用可能な正規表現(任意)
  • C8: MySQL_Regexp: MySQLデータベースに適用可能な正規表現(任意)
  • C9: Oracle_Regexp: Oracleデータベースに適用可能な正規表現(任意)
  • C10: PostgreSQL_Regexp: PostgreSQLデータベースに適用可能な正規表現(任意)
  • C11: SQL_Server_Regexp: SQL Serverデータベースに適用可能な正規表現(任意)
  • C12: Sybase_Regexp: Sybaseデータベースに適用可能な正規表現(任意)
  • C13: Ingres_Regexp: Ingresデータベースに適用可能な正規表現(任意)
  • C14: Informix_Regexp: Informixデータベースに適用可能な正規表現(任意)
  • C15: SQLite3_Regexp: SQLite3データベースに適用可能な正規表現(任意)
  • C16: Teradata_Regexp: Teradataデータベースに適用可能な正規表現(任意)

注:
エクスポートしたcsvファイルは、別のTalend Open Studio for Data Quality内のパターンをインポートするために使用できます。
