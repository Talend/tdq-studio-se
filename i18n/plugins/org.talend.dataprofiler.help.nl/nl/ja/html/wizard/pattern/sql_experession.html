SQL式:

ウィザードの最初のページで、容易に見つかるようにパターンの名前を設定します。

2番目のページで、分析に使用するsqlパターンを設定します。
パターンはシングルクォート"'"で囲む必要があります。
パターン作成後、複数カラム分析エディター内のカラムにドラッグ＆ドロップすることができます。

注:
SQLパターンは正規パターンと異なります。
たとえば、'S'で始まる文字列を検索すると、正規パターンは'S.*'ではなく'S%'です。
SQLパターンはデータベースによるサポートがより容易です:
SQLステートメントの« LIKE »句に表示されます。
SQLパターンのクエリは正規表現のクエリよりも早い場合があります。
ただし、通常はパワーで大幅に劣ります。
詳細は、データベース管理システムのドキュメンテーションを参照して下さい。
