新しいインジケーター

このウィザードにより、ユーザー定義インジケーターが作成されます。

言語を選択し、ウィザードにSQLテンプレートを入力して下さい。これらのキーワードがここでは使用できます:
<%=__WHERE_CLAUSE__%>: データフィルターによって提供されたWhere句を置換します。
<%=__TABLE_NAME__%>: テーブル名を置換します。
<%=__COLUMN_NAMES__%>: カラムを置換します。
<%=__GROUP_BY_ALIAS__%>: "グループ化" whole句を置換します。

ユーザー定義インジケーターを完成するとエディターが開き、ユーザー定義インジケーターのデフォルトカテゴリは"ユーザー定義カウント"になります。
