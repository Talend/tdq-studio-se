Pattern frequency statistics

Indicators in this group determine the most and less frequent patterns.

  • Pattern frequency table: computes the number of most frequent records for each distinct pattern.

  • Pattern low frequency table: computes the number of less frequent records for each distinct pattern.

  • Date pattern frequency table: retrieves the date patterns from date or text columns. It works only with the Java engine.

  • CS Word Pattern Frequency: evaluates the most frequent word patterns. counts the number of record for each distinct pattern. Patterns focus on words and are case sensitive. [Word] represents a word starting with an uppercase character and composed of lowercase characters. [WORD] : represents a word with uppercase characters. [word] represents a word with lowercase characters. [Char] represents a single uppercase character. [char] represents a single lowercase character.

  • CS Word Pattern Low Frequency: evaluates the least frequent word patterns. counts the number of record for each distinct pattern. Patterns focus on words and are case sensitive. [Word] represents a word starting with an uppercase character and composed of lowercase characters. [WORD] : represents a word with uppercase characters. [word] represents a word with lowercase characters. [Char] represents a single uppercase character. [char] represents a single lowercase character.

  • CI Word Pattern Frequency: evaluates the most frequent word patterns. counts the number of record for each distinct pattern. Patterns focus on words and are case insensitive. [word] represents a word, [char] represents a single lowercase character, [alnum] represents an alphanumeric word, [digit] represents a single digit, [number] represents a sequence of digits
